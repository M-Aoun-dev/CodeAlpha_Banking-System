# Banking Management System in C++

A console-based Banking Management System developed in C++ using file handling and structured programming concepts. The project allows users to manage customers, bank accounts, deposits, withdrawals, fund transfers, and transaction history through a simple and interactive interface.

## Features

- Create Customer Accounts
- Create Bank Accounts
- Deposit Money
- Withdraw Money
- Transfer Funds
- View Account Information
- Transaction History Tracking
- View All Customers
- Input Validation
- File Handling for Data Storage
- Beginner-Friendly Console Interface

## Technologies Used

- C++
- File Handling
- Visual Studio / Code::Blocks

## Concepts Used

This project demonstrates the implementation of:

- Structures
- Functions
- Loops
- Conditional Statements
- Arrays and Character Arrays
- String Handling
- File Handling
- Input Validation
- Transaction Management
- Banking System Logic

## How to Run

1. Download or clone the repository
2. Open the project in Visual Studio or any C++ compiler
3. Compile the source file
4. Run the program

## File Description

- `main.cpp` → Contains complete source code
- `customers.txt` → Stores customer records
- `accounts.txt` → Stores bank account details
- `transactions.txt` → Stores transaction history

## Functionalities

### Customer Management
- Add new customers
- Store customer information

### Account Management
- Create saving/current accounts
- Maintain account balances

### Transaction Management
- Deposit money
- Withdraw money
- Transfer funds
- Save transaction records

This project was created for learning and internship practice purposes. It helps beginners understand how banking systems work using basic C++ programming and file handling techniques.
